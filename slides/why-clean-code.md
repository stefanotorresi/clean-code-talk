## Why should I care?

- Coding is hard and expensive
<!-- {_class="fragment fade"} -->

- Productivity decreases over time
<!-- {_class="fragment fade"} -->

- Dirty code quickly becomes a legacy mess
<!-- {_class="fragment fade"} -->

- *The Big Rewrite* <sup><small>(TM)</small></sup> is not a solution

## Why should I write clean code?

- We spend more time reading than writing code
<!-- {_class="fragment fade"} -->

- It should be easy to reason about
<!-- {_class="fragment fade"} -->

- It should welcome change
<!-- {_class="fragment fade"} -->
